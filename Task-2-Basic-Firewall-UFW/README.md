# Task 2: Basic Firewall Configuration

## Objective
Configure UFW on a Linux system to enforce access control policies.

## Configuration Actions
* **Allowed SSH (Port 22):** Enables secure, encrypted remote administration.
* **Denied HTTP (Port 80):** Blocks unencrypted web traffic to prevent unauthorized access and data interception.

## Conclusion
By selectively allowing necessary ports and denying insecure ones, the system's attack surface is significantly reduced.
